## 概要
大学の講義の課題として作成したデバイスドライバで、下記の動作を行います。
- 数字の0~3の入力に応じたLEDの点灯・消灯
- 交通信号機を模したパターンでのLEDの点灯

## 用意・構築環境
- Rasbery Pi 4
- ブレッドボード
- LED
- 抵抗 220Ω
- ジャンパー線
- WSL(Ubuntu 18.04 LTS)
